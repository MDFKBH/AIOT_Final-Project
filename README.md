# AIOT_Final-Project
This is the final project of AIOT lesson.
How to use:
1. Download IP Webcam on your phone and get the IP address.
2. Download IoT MQTT Panel on your phone and add a notification panel.
3. Ensure your computer and two phones are link to the same Wifi network.
4. Run the program, if you see the window jump out. That should be the picture sent from your phone.
5. If there is any person show in the picture, the warning message will jump out on the computer and the phone.
